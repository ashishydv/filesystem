
1. install wampserver from below given link
 https://www.wampserver.com/en/download-wampserver-64bits/

2. unzip and put "filesystem" directory to wamp64/www/ folder.
    Default wamp64 locaton is your c drive.

3. Run installed wampserver.

4. open localhost/filesystem/ in url


to see the fronend code extract filesystem_frontend_code.zip and open src document.
